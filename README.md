# TMS

TMS is a shell that lets you edit terminal settings from your terminal.

As an unintentional bonus, it uses no external utilities - just bash built-ins!

## Commands

(Note: You can also pull these up with `help` in the shell)

|Command|Description|
|-|-|
|`prevcol`|Preview color palettes|
|`prevfmt`|Preview text formats|
|`setcol`|Set a color in the palette|
|`setfmt`|Set a special formatting color|
|`resetcol`|Reset a color to default|
|`resetfmt`|Reset a special color to default|
|`exitsh`|Exit the shell|
